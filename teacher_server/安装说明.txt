========================================
  茶文化课程APP - 教师端服务器
  安装说明
========================================

【安装步骤】

1. 将整个文件夹复制到目标电脑的任意位置（建议：C:\TeacherServer）

2. 首次安装需要初始化数据库：
   - 双击运行 "初始化数据库.bat"
   - 等待提示"数据库初始化完成"

3. 启动服务器：
   - 双击运行 "启动服务器.bat"
   - 或者直接运行 "TeacherServer.exe"

4. 访问Web界面：
   - 打开浏览器
   - 访问：http://localhost:8888

【注意事项】

- 确保电脑已安装 Python 3.8 或更高版本（如果使用 .py 文件）
- 如果使用打包的 .exe 文件，通常不需要安装 Python
- 确保 8888 端口未被占用
- 如需在其他设备访问，请配置防火墙允许 8888 端口

【文件说明】

- TeacherServer.exe       主程序（打包版本）
- run.py                  主程序（Python 源码版本）
- init_db.py              数据库初始化脚本
- config.py               配置文件
- tea_culture.db          数据库文件（首次运行后自动创建）
- uploads/photos/         照片上传目录
- logs/                   日志文件目录

【常见问题】

Q: 无法启动服务器？
A: 检查是否有杀毒软件拦截，或查看控制台错误信息。

Q: 如何修改端口？
A: 编辑 config.py 或 run.py 文件，修改端口号。

Q: 数据库文件在哪里？
A: 数据库文件 tea_culture.db 在程序运行目录下。

Q: 如何备份数据？
A: 复制 tea_culture.db 文件即可。

【技术支持】

如有问题，请查看日志文件：logs/app.log
