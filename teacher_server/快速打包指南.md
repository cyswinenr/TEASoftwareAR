# 快速打包指南

## 最简单的方法（3步完成）

### 1. 安装 PyInstaller

```bash
pip install pyinstaller
```

### 2. 执行打包命令

在 `teacher_server` 目录下打开命令行，执行：

```bash
pyinstaller build.spec
```

或者使用更简单的命令：

```bash
pyinstaller --onefile --console --name=TeacherServer --add-data "app;app" --add-data "config.py;." --add-data "init_db.py;." run.py
```

### 3. 获取打包结果

打包完成后，在 `dist` 目录下会生成 `TeacherServer.exe` 文件。

## 创建安装包

将以下文件复制到一个新文件夹：

```
TeacherServer_安装包/
├── TeacherServer.exe          # 从 dist 目录复制
├── 启动服务器.bat             # 已提供
├── 初始化数据库.bat           # 已提供
└── 安装说明.txt               # 已提供
```

## 安装到目标电脑

1. 将整个 `TeacherServer_安装包` 文件夹复制到目标电脑
2. 首次运行：双击 `初始化数据库.bat`（如果使用 .exe 版本可能不需要）
3. 启动服务：双击 `启动服务器.bat` 或 `TeacherServer.exe`
4. 访问：http://localhost:8888

## 注意事项

- 打包后的 .exe 文件可能被杀毒软件误报，这是正常现象
- 如果打包失败，检查是否安装了所有依赖：`pip install -r requirements.txt`
- 确保目标电脑有足够的磁盘空间（打包后的文件可能较大）
