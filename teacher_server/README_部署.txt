========================================
  茶文化课程APP - 教师端服务器
  快速部署指南（无 Python 环境）
========================================

【重要说明】
本程序已打包为独立可执行文件，无需安装 Python 即可运行！

【快速开始】

1. 首次运行
   - 双击 "启动服务器.bat" 或 "TeacherServer.exe"
   - 程序会自动创建数据库和必要目录

2. 访问服务器
   - 打开浏览器
   - 访问：http://localhost:8888

3. 停止服务器
   - 在控制台窗口按 Ctrl+C
   - 或直接关闭窗口

【文件说明】

- TeacherServer.exe       主程序（包含所有依赖）
- 启动服务器.bat         启动脚本（推荐使用）
- 安装说明.txt           详细说明文档

【自动创建的目录】

程序运行后会自动创建：
- tea_culture.db         数据库文件
- uploads/photos/        照片存储目录
- logs/                  日志目录

【注意事项】

1. 端口占用
   - 确保 8888 端口未被占用
   - 如需修改端口，需要重新打包

2. 防火墙
   - 如果需要在其他设备访问，请允许 8888 端口
   - 打开 Windows 防火墙设置

3. 杀毒软件
   - 打包后的 .exe 可能被误报
   - 添加到白名单即可

4. 数据备份
   - 备份 tea_culture.db 文件
   - 备份 uploads/photos/ 目录

【常见问题】

Q: 程序无法启动？
A: 检查 8888 端口是否被占用，查看控制台错误信息

Q: 如何备份数据？
A: 复制 tea_culture.db 和 uploads 目录

Q: 如何更新程序？
A: 替换 TeacherServer.exe 文件即可，数据库会自动保留

【技术支持】

查看日志文件：logs/app.log
